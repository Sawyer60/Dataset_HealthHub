# 🧠 Data HealthHub — AI Dataset Doctor

An intelligent web application that automatically analyzes, diagnoses, and cleans datasets to make them machine-learning ready in seconds.

Built to simplify data preprocessing for students, analysts, and data scientists.

🌐 **Live Demo:**
https://data-health-hub--harryabraham112.replit.app

---

## 🎯 Project Motivation

Data preprocessing consumes up to **80% of a data scientist’s time**.

This project aims to automate repetitive data cleaning tasks and make dataset preparation accessible to everyone — from beginners to professionals.
